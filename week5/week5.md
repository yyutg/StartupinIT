## Шаг 1. Входная фиксация сегмента и сценария

- **Выбранный сегмент:** Активные криптотрейдеры и DeFi-пользователи в возрасте 22–35 лет, с опытом работы на рынке от 6 месяцев, ежедневно сталкивающиеся с информационной перегрузкой и тратящие 5–10 часов в неделю на ручной сбор данных из 5–10 разных сервисов.
- **Сценарий использования:** Трейдер видит новость о новом проекте или резкое движение цены. Он хочет быстро оценить потенциал актива, но вынужден открывать 5–10 вкладок: DexScreener (графики), CoinGecko (базовая информация), DefiLlama (TVL), блокчейн-обозреватель (контракт), Twitter (новости). Это занимает 15–20 минут, за которые возможность может быть упущена.
- **Основная проблема и последствия:**
  - Разрозненность данных: 68% инвесторов используют 3+ сервиса, 45% жалуются на потерю времени (Binance Research, 2024).
  - Риск скам-проектов: 60% криптоинвесторов хотя бы раз попадали в скам (Chainalysis, 2024).
  - FOMO (страх упустить возможность) приводит к необдуманным решениям и входам на пике.
  - Последствия: потеря времени (5–10 часов в неделю), финансовые потери (500–2000$ в год), упущенная прибыль, эмоциональное выгорание.
- **Текущая замещающая альтернатива:** DexScreener + TradingView + CoinGecko + DefiLlama (комбинация 5–10 сервисов), Telegram-каналы с сигналами, ручной анализ (whitepaper, соцсети).

## Шаг 3. Выделение критериев выбора сегмента

Для активных криптотрейдеров и DeFi-пользователей при выборе платформы для анализа крипторынка ключевыми являются следующие критерии (приоритеты от 1 до 3):

1. **Скорость получения данных** — возможность получать информацию в реальном времени, без задержек (критично для дейтрейдинга).
2. **Глубина анализа** — наличие ончейн-данных, метрик TVL, активности адресов, скоринга проектов.
3. **Единое окно** — все данные в одном интерфейсе, без необходимости открывать 5–10 вкладок.
4. **Цена** — доступная подписка (целевая цена 500–1500₽/мес против $30–500 у Messari).
5. **Надёжность и точность данных** — отсутствие ошибок и задержек, верифицированные источники.
6. **Русский язык** — интерфейс и поддержка на русском языке.
7. **Дополнительные функции** — алерты, портфельный трекер, социальные метрики, отслеживание китов.

**Топ‑3 приоритетных критерия:**
1. Скорость получения данных
2. Глубина анализа
3. Единое окно

## Шаг 4. Формирование гипотезы ценностного предложения

### Основная гипотеза

> **«Для активных криптотрейдеров и DeFi-пользователей, которые тратят 5–10 часов в неделю на анализ проектов и рискуют попасть в скам, CoinTracker обеспечивает единое окно для глубокого анализа крипторынка: графики в реальном времени, ончейн-данные, AI-скоринг проектов и кастомные алерты — за 500–1500₽/мес, сокращая время на анализ с 2–3 часов до 30–40 минут в день и снижая риск попадания в скам-проекты на 70%.»**

### Альтернативные формулировки

1. **Фокус на единое окно:** «Хватит открывать 10 вкладок. CoinTracker — вся аналитика крипторынка в одном месте: графики, ончейн-данные, скоринг, новости».
2. **Фокус на скорость:** «Упускаешь возможности, пока собираешь данные из DexScreener, CoinGecko и DefiLlama? CoinTracker даёт полный анализ проекта за 2–3 минуты».
3. **Фокус на защиту от скамов:** «60% криптоинвесторов хотя бы раз попадали в скам. CoinTracker — AI-скоринг проектов, который отсеивает 90% мусора».

### Ключевое отличие от конкурентов

- **Единое окно** — объединение технической аналитики (графики), ончейн-данных (TVL, активные адреса) и фундаментального анализа (команда, токеномика) в одном интерфейсе. Ни один конкурент не предлагает такой интеграции.
- **AI-скоринг проектов** — автоматическая оценка проекта на основе 20+ метрик (ликвидность, TVL, активность, команда, соцсети). Пользователь видит рейтинг риска и потенциала.
- **Доступная цена** — 500–1500₽/мес против $30–500 у Messari. Ориентация на массового розничного инвестора.
- **Русский интерфейс** — полная локализация для российского рынка, учёт особенностей региональных бирж и сообществ.

### Почему сегмент должен поверить?

1. **Прозрачность скоринга:** Пользователь видит, из каких метрик складывается оценка проекта (ликвидность, TVL, ончейн-активность, команда, соцсети).
2. **Интеграция с биржами:** Возможность торговать прямо из интерфейса через API (Bybit, OKX, HTX).
3. **Ранний доступ:** Запуск в формате закрытой беты с участием известных трейдеров и лидеров криптосообщества.
4. **Freemium-модель:** Бесплатный базовый функционал (базовая информация о монетах) позволяет протестировать сервис перед переходом на платную подписку.
5. **Отзывы и кейсы:** Публикация реальных примеров использования, где CoinTracker помог избежать скама или вовремя войти в проект.

## Шаг 5. Определение ключевого функционала (ядро MVP)

### Core (ядро MVP) — без этих функций ценность не достигается

1. **Дашборд с агрегацией данных по монетам** — единый экран с графиком цен, ончейн-метриками (TVL, активные адреса), фундаментальными данными и новостями.
   *Связь с ценностью:* единое окно — главный критерий выбора.
2. **Графики в реальном времени** — интеграция с DexScreener API, отображение графиков цен и объёмов для DEX и CEX.
   *Связь с ценностью:* скорость получения данных (критично для дейтрейдинга).
3. **Базовый AI-скоринг проектов** — оценка проекта на основе 10–15 метрик (ликвидность, TVL, активность адресов, наличие соцсетей, аудит).
   *Связь с ценностью:* глубина анализа, снижение риска скамов.
4. **Поиск и фильтрация монет** — поиск по названию, фильтры по блокчейну, типу (DeFi, мемкоин, инфраструктура), капитализации, скорингу.
   *Связь с ценностью:* быстрый поиск перспективных активов.
5. **Кастомные алерты** — уведомления о ценовых движениях, изменениях TVL, появлении новых транзакций китов, изменении скоринга.
   *Связь с ценностью:* экономия времени, своевременное реагирование на события.

### Support (повышают удобство и качество)

6. **Портфельный трекер** — отслеживание P&L по всем активам, интеграция с биржами через API.
7. **Социальный модуль** — отслеживание активности топ-трейдеров и китов по публичным кошелькам.
8. **История скоринга** — динамика изменения оценки проекта во времени.
9. **Экспорт данных** — выгрузка отчётов в PDF/Excel.
10. **Расширенные фильтры** — по соотношению риск/доходность, по активности команды, по наличию аудита.

### Later (можно отложить на следующие версии)

11. **Мобильное приложение** (нативные клиенты для iOS/Android) — пока достаточно PWA.
12. **Торговля через интерфейс** — интеграция с API бирж для совершения сделок без переключения окон.
13. **Социальная сеть трейдеров** — публикация идей, копирование сделок.
14. **Институциональные фичи** — расширенная аналитика для хедж-фондов, API для автоматической торговли.
15. **Децентрализованная версия** — анализ без центрального сервера на основе смарт-контрактов.
