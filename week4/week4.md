## Проект: CoinTracker — интеллектуальная платформа для анализа крипторынка

---

### 1. Problem — формулировка потребности

**Заголовок:** Проблема: криптоинвесторы тратят часы на анализ и рискуют попасть в скам-проекты

| Тезис | Описание |
|---|---|
| **Контекст** | На рынке представлено более 2,5 млн криптоактивов, из которых 95% не имеют фундаментальной ценности. Инвесторы и трейдеры используют 3–10 разных сервисов для анализа (DexScreener, TradingView, CoinGecko, DefiLlama), тратя 5–10 часов в неделю на ручной сбор данных. |
| **Боль** | 68% розничных инвесторов используют минимум 3 разных сервиса, 45% жалуются на разрозненность данных и потерю времени (Binance Research, 2024). 60% криптоинвесторов хотя бы раз попадали в скам-проекты (Chainalysis, 2024). Из-за FOMO (страха упустить возможность) трейдеры часто входят на пике или в мошеннические схемы. |
| **Последствия** | Потеря времени (5–10 часов в неделю), финансовые потери от скам-проектов (в среднем 500–2000$ в год на человека), упущенная прибыль из-за входа на хаях, эмоциональное выгорание от информационного шума. |

**Сигналы/основания:**

- 2,5 млн+ криптоактивов на рынке, 95% не имеют ценности (CoinMarketCap, 2025)
- 68% розничных инвесторов используют 3+ сервиса для анализа (Binance Research, 2024)
- 60% криптоинвесторов хотя бы раз попадали в скам-проекты (Chainalysis, 2024)
- Средняя потеря от скам-проекта на одного инвестора — 500–2000$ в год
- Рынок криптоаналитики в РФ — 15–20 млрд ₽ (оценка 2025), рост 30–40% ежегодно
- Количество розничных криптоинвесторов в РФ выросло на 35% за 2024 год (РБК, 2025)

### 2. Customer — целевой клиент и сегмент

**Целевой сегмент:** Активные криптотрейдеры и DeFi-пользователи (22–35 лет)

| Параметр | Значение |
|---|---|
| **Роль** | Пользователь и покупатель в одном лице (сам принимает решение) |
| **Размер сегмента** | ~1,2 млн человек (оценка: 400 тыс. активных трейдеров + 800 тыс. DeFi-пользователей в РФ) |
| **Где проявляется сценарий** | При поиске новых проектов, анализе графиков, проверке ончейн-данных, оценке рисков, отслеживании портфеля |
| **Критерии выбора** | Скорость получения данных (1), глубина анализа (2), единое окно (3), цена (4), надёжность (5) |

**JTBD (Jobs To Be Done):**

> «Когда я вижу новость о новом проекте или резкое движение цены, я хочу за 2–3 минуты получить полную аналитику по активу (графики, ончейн-данные, скоринг, новости), чтобы быстро принять решение о входе, не упуская возможность, несмотря на то что данные разбросаны по 5–10 разным сервисам.»

**Ключевые барьеры:**

- Привычка к бесплатным сервисам (DexScreener, CoinGecko)
- Недоверие к новым платформам («а вдруг данные неточные?»)
- Страх сложного интерфейса («слишком много метрик, не разберусь»)
- Чувствительность к цене (готовы платить 500–1500₽/мес, но не $30–500)

### 3. Market — TAM/SAM/SOM

#### Метод расчёта: комбинированный (top-down + bottom-up)

| Показатель | Значение (₽/год) | Метод расчёта и допущения |
|---|---|---|
| **TAM** | **20 млрд ₽** | Рынок криптоаналитики и инструментов для трейдинга в РФ. *Источник: оценка на основе данных РБК (2025), Binance Research, CoinMarketCap.* |
| **SAM** | **8 млрд ₽** | Доля рынка, приходящаяся на активных трейдеров и DeFi-пользователей (22–35 лет). *Допущение: сегмент составляет 40% от TAM (наиболее активная и платёжеспособная аудитория).* |
| **SOM** | **240 млн ₽** | 3% от SAM. *Допущение: достижимая доля через 3 года при охвате 10% сегмента (120 000 пользователей) и среднем доходе с пользователя 2 000 ₽/год (конверсия в платную подписку).* |

**Визуализация (воронка):**
TAM (20 млрд) → SAM (8 млрд) → SOM (240 млн)

**Детальный расчёт:**

| Параметр | Значение | Источник / Допущение |
|---|---|---|
| **TAM — общий рынок** | | |
| Рынок криптоаналитики и трейдинговых инструментов в РФ | ~20 млрд ₽ | Оценка на основе: количество активных криптоинвесторов в РФ (~3 млн чел) × средние расходы на аналитику (~7000 ₽/год) |
| | | |
| **SAM — доступный рынок** | | |
| Доля сегмента активных трейдеров и DeFi-пользователей | 40% | Наиболее активная и платёжеспособная аудитория (22–35 лет, опыт 6+ месяцев) |
| **SAM** | 20 млрд × 40% = **8 млрд ₽** | Расчёт |
| | | |
| **SOM — достижимый рынок** | | |
| Целевая доля рынка (3 год) | 3% | Реалистичная цель для нишевого стартапа на растущем рынке |
| **SOM** | 8 млрд × 3% = **240 млн ₽** | Расчёт |
| | | |
| **Альтернативный bottom-up расчёт** | | |
| Количество активных криптотрейдеров в РФ | 400 000 чел | Binance Research, РБК (2025) |
| Количество DeFi-пользователей в РФ | 800 000 чел | DefiLlama, Dune Analytics (оценка) |
| **Общая численность сегмента** | 1 200 000 чел | Расчёт |
| Охват сегмента (3 год) | 10% | 120 000 пользователей |
| Конверсия в платную подписку | 8% | Оценка на основе freemium-моделей в FinTech |
| **Количество платных пользователей (3 год)** | 9 600 чел | 120 000 × 8% |
| Средняя цена подписки в месяц | 800 ₽ | Freemium: бесплатно + Pro 500–1500₽/мес |
| **Средний доход с пользователя в год** | 9 600 ₽ | 12 × 800 |
| **SOM (bottom-up)** | 9 600 × 9 600 = **92 млн ₽** | Более консервативная оценка |
| **SOM (скорректированный)** | **150–250 млн ₽** | Диапазон с учётом оптимистичного сценария |

### 4. Why now / Drivers — почему сейчас

| Драйвер | Как влияет на проект |
|---|---|
| **Рост рынка криптовалют** | Капитализация крипторынка превышает $2.5 трлн (2025). Количество розничных инвесторов в РФ выросло на 35% за 2024 год. Растёт спрос на инструменты анализа. |
| **Усложнение рынка** | Появление DeFi, L2-решений, мемкоинов требует более глубокой аналитики. Трейдерам нужны ончейн-данные, метрики TVL, активность адресов — того, чего нет в TradingView. |
| **Уход зарубежных сервисов** | Messari, Nansen и другие международные платформы практически недоступны для РФ из-за санкций и проблем с оплатой. Рынок освободился для локального игрока. |
| **Рост числа скам-проектов** | Chainalysis оценивает потери от криптоскамов в $2–3 млрд ежегодно. 60% инвесторов хотя бы раз попадали в скам. Растёт потребность в надёжных инструментах проверки. |
| **AI-аналитика как тренд** | Появление AI-агентов для анализа блокчейн-данных (Nansen AI, Arkham Intel) подтверждает спрос на автоматизированный скоринг. Пользователи всё больше доверяют AI. |
| **Дефицит единых решений** | Нет платформы, объединяющей техническую аналитику, ончейн-данные и фундаментальный анализ в одном окне с русским интерфейсом и доступной ценой. |
